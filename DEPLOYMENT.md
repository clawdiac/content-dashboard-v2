# ComfyUI Approval Dashboard v2 - Deployment Guide

## Quick Start

### Prerequisites

- Node.js v18 or higher
- macOS, Linux, or Windows with bash
- ComfyUI outputs directory at `~/.openclaw/cache/comfyui-outputs/`

### Installation & Running

```bash
# Clone or navigate to the workspace
cd /Users/clawdia/.openclaw/workspace

# Install dependencies (one-time setup)
npm install

# Build the frontend (required before first run)
npm run build:client

# Start the server
npm start

# Server will run at http://localhost:3000
```

### Development Mode

For development with hot reload:

```bash
# Terminal 1: Start backend with file watching
npm run dev:server

# Terminal 2: Start frontend dev server
npm run dev:client

# Frontend available at http://localhost:5173 (proxies /api to :3000)
```

## Configuration

### Environment Variables

Create a `.env` file in the root (optional):

```bash
PORT=3000                          # Server port (default: 3000)
NODE_ENV=production               # production or development
COMFYUI_OUTPUT_DIR=~/.openclaw/cache/comfyui-outputs  # Watch directory
WORKSPACE_DIR=~/.openclaw/workspace # Feedback storage
```

### File Paths

- **Input:** `~/.openclaw/cache/comfyui-outputs/` (ComfyUI outputs)
- **Feedback JSON:** `~/.openclaw/workspace/.comfyui-feedback.json`
- **Feedback Log:** `~/.openclaw/workspace/skills/comfyui/skill-file.md`
- **Stats Cache:** `~/.openclaw/workspace/.comfyui-stats.json`

## API Endpoints

### REST API

```
GET  /api/images?status=APPROVED&from=2026-02-13&limit=50
GET  /api/file/:filename
POST /api/feedback         { filename, status, feedback_text }
GET  /api/stats
POST /api/batch-feedback   { entries: [...] }
```

See `API_SPEC.md` for full details.

### WebSocket Events

```
image:new          New image detected
image:updated      Image approval status changed
feedback:saved     Feedback saved
stats:updated      Stats refreshed (every 5 seconds)
```

## Troubleshooting

### Port 3000 Already in Use

```bash
# Kill existing process
lsof -ti:3000 | xargs kill -9

# Or use a different port
PORT=3001 npm start
```

### File Watcher Not Picking Up Files

1. Ensure `~/.openclaw/cache/comfyui-outputs/` exists
2. Check file extension is supported (.png, .jpg, .webp, .mp4, etc.)
3. Verify file has fully finished writing (use `awaitWriteFinish`)
4. Check server logs: `tail -f /tmp/comfyui-server.log`

### Feedback Not Saving

1. Ensure `~/.openclaw/workspace/` is writable
2. Check disk space
3. Verify `skills/comfyui/` directory exists:
   ```bash
   mkdir -p ~/.openclaw/workspace/skills/comfyui
   ```

### WebSocket Connection Issues

1. Check server is running: `curl http://localhost:3000`
2. Verify firewall allows port 3000
3. Check browser console for errors (F12)
4. Try different WebSocket transport:
   - Edit `client/src/hooks/useSocket.js`
   - Modify `transports: ['websocket']` to `['polling']`

### Stats Not Updating

1. Ensure feedback is being saved (check `.comfyui-feedback.json`)
2. Verify stats calculation: `curl http://localhost:3000/api/stats`
3. WebSocket should emit `stats:updated` every 5 seconds

## Performance Tuning

### High Image Count (500+)

1. **Virtualization:** Gallery uses `react-virtuoso` for 500+ images
2. **Pagination:** Use `?limit=50&offset=0` in API calls
3. **Lazy Loading:** Images load on-demand
4. **Caching:** Browser caches images with `Cache-Control: max-age=31536000`

### High Feedback Volume

1. Batch feedback with POST `/api/batch-feedback` instead of individual requests
2. Stats cached in `.comfyui-stats.json`
3. File watcher debounces events (200ms)

## Maintenance

### Backup Data

```bash
# Backup feedback
cp ~/.openclaw/workspace/.comfyui-feedback.json ~/.backup/comfyui-feedback.backup.json

# Backup skill file
cp ~/.openclaw/workspace/skills/comfyui/skill-file.md ~/.backup/skill-file.backup.md
```

### Clear Feedback History

```bash
# CAUTION: This deletes all feedback
rm ~/.openclaw/workspace/.comfyui-feedback.json ~/.openclaw/workspace/.comfyui-stats.json
```

### Logs

```bash
# Server logs
tail -f /tmp/comfyui-server.log

# Browser console logs
# Press F12 → Console tab
```
