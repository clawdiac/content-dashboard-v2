# Decisions

Record important choices, their reasoning, and outcomes.

## Format

When you make a big decision:
1. Create a file: `decision-YYYY-MM-DD-title.md`
2. Document: What was decided, why, alternatives considered, outcome
3. Link it to affected projects

## Examples

- **decision-2026-02-13-stack-architecture.md** — Obsidian + QMD + Gemini
- **decision-2026-01-15-target-audience.md** — Who we're building for

---

Record decisions as you make them!
