# Quick Start

Welcome to your Obsidian vault! 👻

This is your single source of truth for all notes, memories, and project context.

## Structure

- **daily/** — Daily notes (auto-created, one per day)
- **projects/** — Major projects (NiggaBets, Social Content Machine, etc.)
- **learnings/** — Lessons learned, mistakes to avoid
- **decisions/** — Important choices and their reasoning
- **tools/** — API keys, configs, SSH hosts, references
- **businesses/** — Business-specific context

## Keyboard Shortcuts

- **Cmd+Shift+D** — Open/create today's daily note
- **Cmd+P** — Command palette (search, create, run commands)
- **Cmd+F** — Search current file
- **Cmd+Shift+F** — Global search across all files

## How It Works With OpenClaw

1. Write notes here in Obsidian
2. QMD watches this vault and indexes everything
3. OpenClaw's memory_search automatically finds relevant context
4. I pull the right information when you ask questions

## Next Steps

1. Start with **MEMORY.md** (your long-term memory)
2. Create files in **projects/** for each major initiative
3. Log daily notes via **Cmd+Shift+D**
4. Link related notes using `[[filename]]` syntax
