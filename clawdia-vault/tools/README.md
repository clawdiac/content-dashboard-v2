# Tools & Configuration

Local setup notes, API keys (reference only), device names, SSH hosts, preferences.

## Files in This Folder

- **api-keys.md** — Reference guide (don't store actual keys here!)
- **devices.md** — Camera names, servers, smart home
- **ssh-hosts.md** — SSH aliases and connection info
- **preferences.md** — Voice settings, model preferences, platform rules

## Security Note

Never commit actual API keys to this vault. Use environment variables or `.env` files instead.

Store references only (e.g., "GEMINI_API_KEY", "OPENAI_API_KEY").
